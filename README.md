# CU-Timepad-Bot
Асинхронный Telegram-бот для мониторинга появления новых записей (слотов) на мероприятия в Timepad. Бот периодически опрашивает страницу события и присылает уведомление, если на заданный период времени вперед (в днях) появились доступные места.

## Установка и запуск
```
echo 'TELEGRAM_TOKEN="YOUR_BOT_TOKEN"' > .env

go install
go run cmd/bot
```
